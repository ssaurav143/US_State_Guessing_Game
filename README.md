# US States Guessing Game

An interactive geography game built using Python Turtle and Pandas.  
Test your knowledge of US states by guessing their names and seeing them appear on the map.

---

## 🎮 Features

- Interactive US map using Turtle graphics  
- Input-based guessing system  
- Correct guesses are displayed on the map  
- Tracks number of correct answers  
- Prevents duplicate guesses  
- Exports missed states to a CSV file for learning  

---

## 🧠 How It Works

- The game displays a blank map of the United States  
- You enter state names one by one  
- If correct, the state name appears at its location  
- The game continues until all 50 states are guessed or you exit  
- If you exit early, a file with missing states is generated to learn if any states are missing

---

## 🎯 How to Play

1. Run the program  
2. Enter the name of a US state  
3. Watch it appear on the map if correct  
4. Type `Exit` to quit early
